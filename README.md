🛒 Brainwave Store – Online Store Homepage

🌐 Live Demo
🔗 https://sahoo2025.github.io/online-store/

📌 Project Overview
Brainwave Store is a modern and responsive online store homepage built using HTML, CSS, and JavaScript.
It showcases products in different categories with interactive features like filtering, sorting, and search.
This project demonstrates UI/UX design, frontend development, and dynamic data handling using JSON.

🚀 Features
✔️ Product Categories (Electronics, Fashion, Books)
✔️ Dynamic Product Cards with Images, Price & Ratings
✔️ Search Functionality 🔍
✔️ Category Filtering 🎯
✔️ Price Sorting (Low → High / High → Low)
✔️ Responsive Design (Mobile + Desktop) 📱💻
✔️ Smooth Hover Effects & Modern UI

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & Responsive Design
JavaScript (ES6) – Logic & Interactivity
JSON – Product Data

🌍 Deployment
This project is deployed using GitHub Pages.
To deploy:
Push code to GitHub
Go to Settings → Pages
Select branch: main
Save and get live link
